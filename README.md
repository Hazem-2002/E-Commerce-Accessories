A dynamic front-end E-Commerce application built using HTML5, CSS3, Bootstrap, and JavaScript.

---

## Key Features

### User Authentication
- Registration & Login system
- User data stored in LocalStorage
- Persistent sessions and personalized experience
- Dynamic switching between Login & Register modals

### Product Management
- Product cards displaying image, name, price, and category
- Add to Cart & Add to Favorites functionality
- Login validation before performing actions

### Cart System
- Dynamic cart notification badge
- Real-time product updates
- Quantity control using (+ / -) buttons
- User-specific cart storage
